About me

Oft-grumpy software engineer. Living on Gadigal land / Glorious Inner West Sydney.

Current work is in energy market software, data and analysis. Pretty keen on that whole energy transition! Previous work has been embedded software / consumer electronics.

Dad to two kids. Likes a good bushwalk, motorbikes, cooking, tinkering with electronics.

Full-time remote WFH crazy cat gentleman.

I mostly work on software.

I have lots of experience at startup-type companies, working across the whole product stack from design, development, testing to deployment and shipping products.

I’ve done loads of embedded stuff: C/C++, ARM assembly, RTOSes, microkernels, FPGAs, device drivers, audio codecs, DSPs, Linux, bluetooth, WiFi, networking.

I’ve done Android apps with both Java and Kotlin. I’m excited by functional programming too.

I’ve done server backends in Python and Go, RESTful API design, websockets, real time apps. Recently I’ve been doing a lot more backend C# and .NET.

I’m no stranger to Windows native development with .NET and Windows Presentation Framework, XAML and so on.

I know enough Javascript and front end web technology to be dangerous, and I’m happy to dive into the latest framework of the day.

I’ve used databases extensively: MS SQL, Postgres, MySQL, RethinkDB, Realm, CouchDB, you name it.

I’ve done ops-y things, deployed and monitored servers on AWS and other cloud services. Terraform and other infrastructure-as-code things are a great idea.

I know which end to hold a soldering iron and can read an electronics hardware schematic.

I’ve led software teams, mentored, code reviewed, done some project management and represented companies at trade shows.

I like working on things that make a difference, making a positive impact on the world while broadening my experience with new technology.

I reluctantly have a LinkedIn.

I post (mostly dad jokes) on the fediverse at @akent@aus.social. You can also find me on BlueSky.

I have a few (mostly abandoned) side coding projects at github.com/akent.

I worked on some patents.

I did Computer Science at Sydney University. I also have an Arts degree in Politics and Journalism. Don’t hold that against me, necessarily!

I like solving problems in the most sensible and obvious way possible.

I want to solve your problems, too.

I’m not currently available for any extra work but you can email me, happy to chat: a@akent.org
